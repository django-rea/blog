# Kamasi Blog

## Getting started

To set up your environment to develop on this blog, you need to install [jekyll](https://jekyllrb.com/docs/installation/)
After installing globally jekyll:
- clone this repo
- run `jekyll serve`
- open your browser at `http://localhost:4000`
